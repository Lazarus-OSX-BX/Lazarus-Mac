## Frequently Asked Questions (FAQ)

**What makes Lazarus different from alternatives?**
Lazarus combines Free Pascal integrated development environment with deep macOS integration, providing cross-platform Pascal IDE through optimized performance and native system features.

**Is Lazarus compatible with Apple Silicon?**
Yes, Lazarus is built as a universal binary supporting both Intel and Apple Silicon processors with optimized performance for each architecture.
